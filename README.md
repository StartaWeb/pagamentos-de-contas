# Controle de Pagamentos - StartWeb

Uma aplicação web profissional para o registro, controle e pesquisa de pagamentos de contas. Este sistema foi desenvolvido para ser estático (sem dependência de banco de dados, utilizando apenas o armazenamento local do navegador - `localStorage`), sendo ideal para publicação no **GitHub Pages**.

## Funcionalidades
- **Cadastro de Contas:** Insira o código de barras, data de vencimento, descrição e valor da conta.
- **Filtro e Pesquisa:** Filtre relatórios/documentos por código de barras, valor, descrição ou período (datas).
- **Relatórios Semanais:** Resumo visual das contas organizadas por semana.
- **Armazenamento:** Persistência de dados completa no navegador.
- **Design:** Moderno e profissional utilizando Glassmorphism e ícones SVG.
- **Repositório Hospedado:** Desenvolvido para rodar facilmente em *https://github.com/StartaWeb/pagamentos-de-contas.git*

## Como Usar
Para usar a aplicação localmente:
1. Clone o repositório ou baixe os arquivos.
2. Abra o arquivo `index.html` em qualquer navegador web moderno.
3. Não é necessário nenhum servidor ou configuração de banco de dados.
